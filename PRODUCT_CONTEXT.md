- **Framework:** Next.js 14 App Router (SSG — Static Site Generation)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (useScroll, useTransform, useInView, AnimatePresence)
- **Font:** Heebo (Google Fonts), משקלים 300–900
- **אייקונים:** Lucide React
- **שפה:** RTL עברית (`<html lang="he" dir="rtl">`)
- **Image optimization:** Next.js Image component (מופעל)
- **Analytics:** Google Tag Manager placeholder (GTM-XXXXXXX — צריך להחליף)

### צבעי Brand
- **רקע:** `#080808` (כמעט שחור)
- **זהב ראשי:** `#F5A624`

## שפה ואסטרטגיית תוכן

- **גוף:** זכר, יחיד, גוף שני ("אתה", "שלך", "תבין")

- **אין:** מונחים מורכבים בלי הסבר, הבטחות לא מציאותיות, לחץ אגרסיבי
- **יש:** Urgency אמיתית (ספירה לאחור), Social proof, Guarantee ברורה, Value stack מפורט
- **אסטרטגיה:** AIDA — Attention (hero) → Interest (narrative+problem) → Desire (solution+proof) → Action (decision)
